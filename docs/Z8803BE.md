# ZBTLink ZBT-Z8803BE notes

This folder is based on the Z8803BE OpenWrt 25.12 research build environment.

Important known hardware details:

- SoC: MediaTek MT7988A
- RAM: 1 GB
- Storage: eMMC on the user's device
- Wi-Fi: MT7990 / MT7991
- Modem: Qualcomm X62 over USB-QMI
- LAN: lan0, lan1, lan2
- WAN: eth1
- SFP/WAN: eth2 as wan_sfp

The workflow copies the Z8803BE DTS and board files into a fresh OpenWrt checkout, injects the device block, installs the saved config seed, and builds firmware in GitHub Actions.
